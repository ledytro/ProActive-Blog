Primeros pasos en Git y GitHub

¿Qué es Git?

Git es un sistema de control de versiones (VCS) distribuido que permite llevar un registro de los cambios en archivos y directorios de un proyecto. Almacena la historia completa de todas las revisiones y permite revertir cambios y trabajar en ramas de desarrollo paralelas.

¿Qué es GitHub?

GitHub es una plataforma en línea que utiliza Git para almacenar y compartir proyectos de software. Además de almacenar el código, también proporciona herramientas para colaborar en proyectos, realizar seguimiento de errores y discutir cambios.

¿Qué es un VSC?

Un VCS es un sistema de control de versiones, es una herramienta que permite llevar un registro de los cambios realizados en un proyecto y facilita la colaboración en equipo. Git y GitHub son solo dos ejemplos de sistemas de control de versiones y plataformas para compartir proyectos, hay otros como SVN, Mercurial, etc.
